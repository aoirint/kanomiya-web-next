# kanomiya-web-next

## Dependencies

- Node 20 (>= 20.7.0)
- npm 10

### Switch Node version with nvm

- <https://github.com/nvm-sh/nvm>
- <https://github.com/coreybutler/nvm-windows>

```shell
nvm use
```

### Install node_modules

```shell
npm ci
```

## Preview

```shell
npm run dev
```

## Build

```shell
npm run build

serve out/
```
