# kanomiya.com-next

## Dependencies

- Node 18 (>=18.14.2)
- npm 9

### Switch Node version with nvm

- <https://github.com/nvm-sh/nvm>
- <https://github.com/coreybutler/nvm-windows>

```shell
nvm use
```

### Install node_modules

```shell
npm ci
```

## Preview

```shell
npm run dev
```

## Build

```shell
npm run build

serve out/
```
